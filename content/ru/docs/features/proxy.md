# Прокси
Бот поддерживает коммуникацию через HTTP/Socks5 прокси. Для того чтобы включить данную функциональность необходимо отредактировать файл appsettings.json, добавив секцию Bot.Proxy

## Параметры

Блок Bot.Proxy состоит из следующих параметров:

1. **Type** - тип прокси. Поддерживаемые значения: "Http", "Socks5". Параметр обязательный.
2. **Address** - адрес прокси-сервера. Параметр обязательный.
3. **Port** - порт прокси-сервера. Параметр обязательный.
4. **Username** - логин для аутентификации (опционально).
5. **Password** - пароль для аутентификации (опционально).

## Пример конфигурации

В данном примере устанавливается подключение без аутентификации через Socks5 прокси.

{{< details "appsettings.json" open >}}
```
{
    "Bot": {
        "Proxy": {
            "Type": "Socks5",
            "Address": "test.server.com",
            "Port": 1080, 
        }
    }
}
```
{{< /details >}}

## Известные проблемы
- Подключение через прокси может быть значительно медленнее соединения напрямую. При этом в случае разрывов соединения доставка уведомлений не гарантируется.
